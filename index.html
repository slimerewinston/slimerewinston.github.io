How to Install on iOS

  1. Create a Bookmark: Bookmark *any* page in Safari (like this one).
  2. Copy the Code: Copy the code block below:

  4. Paste the URL: Change the name to "FCWatch Overhaul" and paste the code into the **URL** field. Save it.
  5. Launch: Go to fc-watch.com, open your bookmarks, and tap **FCWatch Overhaul**.
