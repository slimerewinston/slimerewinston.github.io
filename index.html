Ultimate iOS Edition

Install the mobile plugin via Safari Bookmarklet. This new V2 engine injects the latest updates directly from GitHub Pages and persists automatically across site navigations.

javascript:(function(){var s=document.createElement('script');s.src='https://slimerewinston.github.io/FCWatch-Overhaul/bookmarklet/fcwatch-overhaul-mobile.js?v='+Date.now();document.head.appendChild(s);})();

🛠 How to Install on iPhone

  1. Tap the Share button (square with arrow) at the bottom of Safari and select Add Bookmark. Name it "FCW Mobile" and tap Save.
  2. Tap the Copy Code button above.
  3. Tap the Bookmarks button (the book icon) at the bottom of Safari.
  4. Tap Edit at the bottom right.
  5. Tap the "FCW Mobile" bookmark you just made.
  6. Tap the URL field, clear it, and Paste the code you copied. Tap Done.
  7. Visit any FC Watch page, tap your Bookmarks icon, and tap "FCW Mobile" to launch the overhaul!
